# Gas_Monitoring
Gas Monitoring Program by WPF Framework - Acquired GS certification

최대 5개 가스 취득기의 가스 종류와 농도를 실시간 및 기록 그래프로 모니터링하는 프로그램.

TTA에서 시험하는 'GS인증 1등급' 획득함.

WPF Framework - MVVM 패턴으로 개발됨.
